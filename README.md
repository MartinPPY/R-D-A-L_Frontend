# R-D-A-L Frontend

Aplicacion web hecha con React y TypeScript, ademas de dependencias como react-hook-form, react query, shadcn/ui y tailwindcss.

## Instalacion

### Requisitos:

- Tener Node JS instalado.

1. Clonar el repositorio
2. Ejecutar ` npm install `
3. Si hay vulnerabilidades ejecutar ` npm audit fix `

- Para ejecutar la aplicacion en modo desarrollo ejecutar: ` npm run dev `
- Para buildear la aplicacion ejecutar ` npm run build ` 

! Es importante que el backend este operativo para que la aplicacion web funcione correctamente.

Seguramente existiran 2 usuarios:

- alumno.rda con su credencial: Choco.123
- moderador.rda con su credencial: Choco.123
